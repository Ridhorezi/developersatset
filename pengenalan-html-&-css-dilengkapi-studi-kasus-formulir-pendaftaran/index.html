Pengenalan HTML & CSS dilengkapi studi kasus Formulir Pendaftaran.

July 8, 2023 · 4 min read
DeveloperSatSet
DSI

Berikut pengenalan satset HTML & CSS dilengkapi studi kasus Formulir Pendaftaran.

Pengenalan HTML & CSS​

  • HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membangun struktur dan mengatur konten dari sebuah halaman web. HTML menggunakan elemen-elemen tag untuk memberikan instruksi kepada browser tentang bagaimana halaman web harus ditampilkan. Setiap elemen tag memiliki fungsi tertentu dan dapat digunakan untuk menyusun judul, paragraf, gambar, tautan, tabel, formulir, dan lain sebagainya.
<html>  : Mengawali dokumen HTML.
<head>  : Mendefinisikan informasi mengenai dokumen.
<title> : Menentukan judul dokumen yang ditampilkan di bilah judul browser.
<body>  : Memuat seluruh isi halaman web.
<h1>    : Menandai tingkatan dan ukuran teks dari judul.
<p>     : Membuat paragraf teks.
<a>     : Membuat tautan atau hyperlink.
<img>   : Menampilkan gambar di halaman web.
<table> : Membuat tabel untuk menyusun data dalam bentuk baris dan kolom.
<form>  : Membuat formulir untuk mengumpulkan data pengguna.
<input> : Menyediakan kotak input untuk pengguna memasukkan data.
  • CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan tata letak dari elemen-elemen HTML. Dengan menggunakan CSS, Anda dapat mengubah warna, ukuran, jenis huruf, jarak antar elemen, dan banyak lagi. CSS memisahkan antara struktur (HTML) dan desain (CSS) sebuah halaman web, sehingga memungkinkan pemisahan tugas antara pengembang web dan desainer web.
body {
  font-family: Arial, sans-serif;
  background-color: #f2f2f2;
}
    
h1 {
  color: #333;
}
    
p {
  font-size: 16px;
  line-height: 1.5;
}

Menggabungkan HTML dengan CSS​

<!DOCTYPE html>
<html>
<head>
  <title>Contoh Halaman Web</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      background-color: #f2f2f2;
    }
    
    h1 {
      color: #333;
    }
    
    p {
      font-size: 16px;
      line-height: 1.5;
    }
    
    .highlight {
      background-color: yellow;
    }
  </style>
</head>
<body>
  <h1>Ini adalah contoh halaman web</h1>
  <p>Ini adalah contoh paragraf dengan <span class="highlight">teks yang dihighlight</span>.</p>
</body>
</html>

Studi Kasus Form Pendaftaran​

  1. Buat file HTML baru dan buka di editor teks.
  2. Di dalam elemen body, buat elemen form dengan atribut action dan method. Atribut action menentukan ke mana data formulir akan dikirim, dan atribut method menentukan cara pengiriman data.
<form action="/submit-form" method="POST">
  <!-- Isi formulir akan ditambahkan di sini -->
</form>
  1. Untuk setiap bidang yang ingin diisi oleh pengguna, buat elemen label. Setiap elemen label harus memiliki atribut for yang sesuai dengan id elemen input yang sesuai.
<label for="nama">Nama:</label>
<input type="text" id="nama" name="nama" placeholder="Masukkan nama Anda" />
  1. Di dalam setiap elemen label, buat elemen input dengan atribut type, id, name, dan placeholder. Atribut type menentukan jenis input (misalnya "text", "email", "password", dll.), atribut id menyediakan pengidentifikasi unik untuk input, atribut name menentukan nama bidang input, dan atribut placeholder menyediakan petunjuk kepada pengguna tentang apa yang harus dimasukkan dalam bidang tersebut.
<label for="email">Email:</label>
<input type="email" id="email" name="email" placeholder="Masukkan alamat email Anda" />
  1. Tambahkan elemen input dengan atribut type = submit untuk membuat tombol Submit yang memungkinkan pengguna mengirimkan informasi formulir.
<input type="submit" value="Submit" />
  1. Gunakan CSS untuk mengatur tata letak dan tampilan elemen formulir agar terlihat menarik secara visual.
<style>
  form {
    /* Gaya tata letak formulir */
  }

  label {
    /* Gaya tampilan label */
  }

  input {
    /* Gaya tampilan input */
  }

  input[type="submit"] {
    /* Gaya tampilan tombol submit */
  }
</style>

Kode Lengkap & Output Form Pendaftaran​

<!DOCTYPE html>
<html>
  <head>
    <title>Formulir Pendaftaran</title>
  </head>

  <body>
    <h1>Formulir Pendaftaran</h1>
    <form action="/submit-form" method="POST">
      <label for="nama">Nama:</label>
      <input type="text" id="nama" name="nama" placeholder="Masukkan nama Anda" />
      
      <label for="email">Email:</label>
      <input type="email" id="email" name="email" placeholder="Masukkan alamat email Anda" />
      
      <label for="password">Password:</label>
      <input type="password" id="password" name="password" placeholder="Masukkan password Anda" />
      
      <input type="submit" value="Submit" />
    </form>
  </body>

  <style>
      form {
        margin: 20px;
        padding: 20px;
        border: 1px solid #ccc;
        border-radius: 5px;
        background-color: #f2f2f2;
      }
    
      label {
        display: block;
        margin-bottom: 10px;
        font-weight: bold;
      }
    
      input {
        width: 100%;
        padding: 5px;
        margin-bottom: 10px;
        border: 1px solid #ccc;
        border-radius: 3px;
      }
    
      input[type="submit"] {
        background-color: #4caf50;
        color: white;
        cursor: pointer;
      }
    </style>
</html>
