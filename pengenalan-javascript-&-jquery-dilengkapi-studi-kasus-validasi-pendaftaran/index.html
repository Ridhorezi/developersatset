Pengenalan JavaScriptâ

JavaScript adalah bahasa pemrograman yang populer dan sering digunakan untuk mengembangkan aplikasi web interaktif. Dalam konteks pengembangan web, JavaScript digunakan untuk memberikan interaksi dan fungsionalitas dinamis pada halaman web. Berikut adalah beberapa poin penting tentang JavaScript :

  • JavaScript adalah bahasa pemrograman yang berjalan di sisi klien (client-side) pada browser web. Ini berarti bahwa kode JavaScript dieksekusi oleh browser pada komputer pengguna, bukan di server.
  • JavaScript dapat digunakan untuk mengubah konten HTML, mengatur perilaku halaman web, dan berinteraksi dengan pengguna melalui elemen-elemen interaktif seperti formulir, tombol, dan tautan.
  • Dalam JavaScript, kita dapat mengambil dan memanipulasi elemen HTML melalui Document Object Model (DOM). DOM merupakan representasi struktur hierarkis dari elemen-elemen HTML pada halaman web.
  • JavaScript juga mendukung berbagai fitur pemrograman seperti variabel, tipe data, operator, kontrol alur, fungsi, dan objek. Ini memungkinkan kita untuk membuat kode yang kompleks dan terstruktur.

Contoh Variabel & Tipe Data :

// Deklarasi variabel
let name = "John";
const age = 25;

// Tipe data
let number = 10;
let text = "Hello";
let isTrue = true;
let fruits = ["apple", "banana", "orange"];
let person = { name: "John", age: 25 };

Contoh Operator :

let x = 5;
let y = 3;

let sum = x + y;
let difference = x - y;
let product = x * y;
let quotient = x / y;
let remainder = x % y;

let isGreater = x > y;
let isEqual = x === y;

Contoh Pengendalian Alur :

if (age >= 18) {
  console.log("Anda sudah dewasa");
} else {
  console.log("Anda masih anak-anak");
}

switch (day) {
  case "Monday":
    console.log("Hari Senin");
    break;
  case "Tuesday":
    console.log("Hari Selasa");
    break;
  default:
    console.log("Hari lainnya");
    break;
}

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

let numbers = [1, 2, 3, 4, 5];

for (let number of numbers) {
  console.log(number);
}

Contoh Fungsi :

function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("John");

function calculateSum(a, b) {
  return a + b;
}

let result = calculateSum(5, 3);
console.log(result);

Contoh Iterasi dan Array :

let fruits = ["apple", "banana", "orange"];

fruits.forEach(function(fruit) {
  console.log(fruit);
});

let filteredFruits = fruits.filter(function(fruit) {
  return fruit.length > 5;
});

console.log(filteredFruits);

Pengenalan Jqueryâ

jQuery adalah pustaka JavaScript yang populer dan kuat yang menyederhanakan manipulasi dan interaksi dengan DOM dalam pengembangan web. Ini menyediakan cara yang lebih mudah dan singkat untuk melakukan tugas-tugas umum seperti pemilihan elemen, penanganan event, animasi, dan komunikasi dengan server. Berikut adalah beberapa poin penting tentang jQuery :

  • jQuery adalah pustaka JavaScript yang ringan dan kompatibel dengan berbagai browser web.
  • Dalam jQuery, pemilihan elemen pada halaman web dapat dilakukan menggunakan sintaks yang mirip dengan CSS. Ini memungkinkan kita untuk mengambil dan memanipulasi elemen dengan mudah.
  • jQuery menyediakan berbagai metode dan fungsi yang mempermudah manipulasi DOM, animasi, dan penanganan event. Misalnya, kita dapat menggunakan metode hide() untuk menyembunyikan elemen, atau metode fadeIn() untuk membuat elemen muncul secara perlahan.
  • Selain itu, jQuery juga menyediakan utilitas AJAX yang memudahkan komunikasi dengan server untuk mendapatkan atau mengirim data tanpa perlu me-refresh halaman secara keseluruhan.

Berikut adalah contoh penggunaan jQuery untuk menyembunyikan dan menampilkan elemen dengan animasi :

<!DOCTYPE html>
<html>
<head>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.0/jquery.min.js"></script>
  <title>Contoh penggunaan jQuery untuk menyembunyikan dan menampilkan elemen dengan animas</title>
</head>
<body>
 <button id="myButton">Klik Saya</button>
  <div id="myDiv">Teks yang akan disembunyikan dan ditampilkan</div>
</body>
<script>
$(document).ready(function() {
  $('#myButton').click(function() {
    $('#myDiv').toggle('slow');
  });
});
</script>
</html>

Studi Kasus Validasi Formulir dengan JavaScriptâ

Berikut adalah contoh studi kasus penggunaan JavaScript untuk melakukan validasi formulir, di mana kita akan memeriksa apakah bidang-bidang tertentu telah diisi sebelum mengirimkan data ke server :

<!DOCTYPE html>
<html>
<head>
  <title>Contoh Pop Up Javascript</title>
</head>
<body>
  <form id="myForm" onsubmit="return validateForm()">
  <label for="name">Nama:</label>
  <input type="text" id="name" style="margin-bottom: 5px;">
  <br>
  <label for="email">Email:</label>
  <input type="email" id="email">
  <br>
  <button type="submit" style="margin-left: 12%; margin-top: 1%;">Kirim</button>
</form>
</body>
<script>
   function validateForm() {
    const name = document.getElementById('name').value;
    const email = document.getElementById('email').value;

    if (name === '' || email === '') {
      alert('Harap isi semua bidang yang diperlukan!');
      return false;
    }

    // Proses pengiriman data ke server...
  }
</script>
</html>
